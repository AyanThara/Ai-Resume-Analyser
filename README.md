# AI Resume Analyzer

AI Resume Analyzer is a web application that helps users evaluate resumes against job descriptions using AI-powered feedback and ATS-style scoring.

## Features

- Google Authentication
- OTP Authentication
- Resume Upload (PDF)
- Resume Storage
- ATS Score Generation
- AI-Based Resume Feedback
- Job Description Matching
- Responsive Modern UI

## Tech Stack

### Frontend
- React
- TypeScript
- React Router v7
- Tailwind CSS
- Zustand

### Backend & Cloud Services
- Puter.js
- Puter Authentication
- Puter File Storage
- Puter Key-Value Database
- Puter AI

## Project Workflow

1. User logs in using Google or OTP.
2. User uploads a resume in PDF format.
3. Resume is stored securely in cloud storage.
4. AI processes the resume and compares it with the provided job description.
5. ATS score and feedback are generated.
6. Results are displayed in an easy-to-read dashboard.

## Installation

Clone the repository:

```bash
git clone https://github.com/AyanThara/Ai-Resume-Analyser.git
```

Navigate to the project:

```bash
cd Ai-Resume-Analyser
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```
